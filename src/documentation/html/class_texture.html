Detailed Description

This class abstracts a Texture. Usually a Texture is generated by the TextureManager class. If you want to create a Texture by you're own, be sure to delete the Texture by youself and NOT by using the deleteTexture() function of the TextureManager class. Actually there is no need to create Textures by not using the TextureManager. If you want to directly write into the TextureData, use the PixelBuffer Object. Pixelbuffers are able to create a valid Texture Object from their Buffer which you can use as a normal texture.
